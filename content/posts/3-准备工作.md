## 二、项目结构

```bash
vue-demo
	|--- node_modules
	|--- public
	|     |--- favicon.ico    # 浏览器标题栏的小图标
	|     |--- index.html     # 项目页面
	|--- src
	|	  |--- assets 		 # 静态资源文件，例如图片、字体图标等
	|	  |--- components	 # 组件
	| 	  |--- App.vue		 # 最外层容器组件
	| 	  |--- main.js 		 # 项目入口文件
```
